  1. Add a New Book

    • Enter the book's ISBN number. If you have a barcode scanner at hand, click into the text box and then scan the book. If you don't have a scanner, you'll have to type it manually and then click "Add Entry". The code should recognize both ISBN10 and ISBN13, but if one doesn't work, try the other. If your book does not have an ISBN, there is a link below that field to enter a book manually.
    • You should now be directed to a form. If you entered an ISBN number, then many of the fields will be filled out for you. It's still a good idea to double-check this data before submitting the form. You'll also need to add any information that is missing (including SC specific data like purchase date and price).
    • Once you've filled out the form with all of the required data, click the "Save" button at the bottom of the page. Once you do that, you'll get a barcode number for the book. If you need to print out a new barcode, see "Print Out Barcodes" below.

  2. Edit a Book

    • Enter a search query in the text box below "Search for an entry to edit below:", then click the "Search" button below. You should see a list of books that match that search query. Click on the book you wish to edit. (Tip: You can also get to the edit entry form by clicking the small pencil icon in the upper-right-hand corner, assuming you're logged in as an admin.)
    • You should now be directed to a form. The existing data for the book you clicked on should be filled in. Make the changes you wish to make, then click the "Save" button at the bottom of the form.
    • Back to Top

  3. Print Out Barcodes

    • In the reports section in the bottom-right-hand corner of the admin dashboard, click "Generate New Barcodes." You will be directed to a new page with instructions on how to create barcodes.
    • To generate a single barcode, enter the last five digits of the barcode number in the first text box. Your browser will then download a PNG image file with the barcode on it.
    • To generate a range of barcodes, enter the last five digits of the first barcode number in the range in the first text box and the last five digits of the last barcode number in the range in the second text box. Your browser will then download a PNG image file with each barcode on it.
    • To print out your barcode(s), follow the instructions given for the label paper you're using.

  4. Export the Database

    • In the reports section in the bottom-right-hand corner of the admin dashboard, click "Export." Your browser will then download a JSON file. Save this file somewhere you'll be able to find it later in case you need it.
    • You should download a backup copy of the database several times a year, and especially after any major edits or additions, so that you always have a copy that's reasonably up-to-date just in case the website breaks.
    • Back to Top

  5. Import the Database

    • In the reports section in the bottom-right-hand corner of the admin dashboard, click "Import." This will then bring up a file selection window. Locate your most recent copy of the database - it should be a file called "database [date].json" - and select it to upload it to the website.
    • This will then pop up a dialog window with some stats about what will change from the existing version of the database. If there is a large discrepancy between the existing copy and your backup, please be very careful about overwriting the database! If you want to keep the database as is, click the button that says "No, take me back."
    • If you do want to overwrite the existing database with your backup, click the button that says "Yes, import this data." The upload process may take up to 15-20 seconds, so be patient (though it will time out if it takes longer than 30 seconds). When it finishes, you should see a popup telling you that the database has been uploaded successfully.
